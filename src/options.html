BucketSpy Config

Manage S3 request monitoring settings

Basic Settings

S3 Request Monitoring

Enables monitoring of S3 requests. Disabling will stop request detection and badge display.

Badge Display

Displays a badge with the number of requests on the extension icon.

Monitoring Targets

This extension monitors S3 requests that match the following pattern:

https://*.s3-*.amazonaws.com/*

e.g. https://mybucket.s3-ap-northeast-1.amazonaws.com/path/to/file.jpg

Privacy Policy

This extension saves information about detected S3 requests locally and does not send it externally. The request history is cleared when the browser is closed or the page is refreshed.
